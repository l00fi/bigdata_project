# Список сервисов

- prefect (http://localhost:9090) - дашборд для запуска пайплайнов;
- spark-master (http://localhost:8080) - мониторинг кластера;
- minio (http://localhost:9001) - дашборд хранилища (озеро данных) сырых данных;
- grafana (http://localhost:3000) - лашборд для аналититки по данным.
